Search Filter Example 2

In this example, I'm required to input my country into a text box, and a filter panel appears. If required, the user can choose from one of the options, in one of two ways:

  • On desktop, they can use the down arrow key to move into and through the list. Using the up arrow will move up through the list and back onto the input field. Pressing the enter key will select the chosen option. (For debate) If focus is on one of the options and I press the tab key, what happens?
  • On mobile (for discussion), the user can swipe right whilst the list is shown, and move through the list. Double tapping will select the item that currently has focus.

If the user starts typing into the search filter text box and there are more than two characters, the list of countries appears and is filtered to remove countries that don't contain the character string.

If no countries contain the string entered, then the list will disappear.
