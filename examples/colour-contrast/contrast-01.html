Checking Colour Contrast

Black on white, all elements have colours specified

Here, the container and all the elements within the container have specific foreground and background colours set. Using black and white this should give the maximum possible colour contrast.

Black on grey, all elements have colours specified

Here, the container and all the elements within the container have specific foreground and background colours set. Using black text on a dark grey background the colour contrast is in the balance.

Red on grey, container specified, elements not specified

In this block, the container has foreground and background colours specified, but the elements within the block have no colours specified. So they inherit forground colour from the container. The colour combination does not have enough contrast.

Black on white, container specified, elements not specified

Here, the container has background and foreground colours specified, but once again, the elements within the container do not have colours specified - so they will inherit forground colour from the container. Black on white has very good colour contrast.

White text sitting on semi-opaque background on image

Here, the container has a background image specified, elements within the container have colours specified including semi-transparent background.
